# ①課題名
1日3つ ×3分 × 着手にフォーカスしたタスク管理アプリ

## ②課題内容（どんな作品か）
- 1日にフォーカスするタスクを3つだけ自分で設定し、
「まず3分だけ取りかかる」ことに特化して設計したタスク管理アプリ。
コンセプトは 「5秒で動き出し、3分だけやってみる。」

🔸使用方法
① タスクの入力
画面上に3つの入力欄が表示されます。
その日に取り組むタスクを最大3つまで設定します。
入力後、「Start Task 1」ボタンを押すことで着手フローがスタートします。

② 5秒カウントダウン（着手スイッチ）
スタートを押すと、まず5秒のカウントダウンが開始されます。
5秒は、「とりあえず動き始める」ための時間としてカウントしています。

③ 180秒（3分）タイマー
5秒が終了すると、自動で180秒（3分）のカウントがスタートします。
3分やればokのルール化をすることで着手までのハードルを低くする設計です。
